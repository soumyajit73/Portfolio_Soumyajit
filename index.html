Projects These are some of my projects that i have worked on as backend developer.

Shabdh.AI

Shabdh.AI is an innovative solution designed to bridge the communication gap between the deaf and hard-of-hearing community and the hearing population. It translates Indian Sign Language (ISL) into text and speech in real time, enabling seamless communication. The system leverages deep learning and computer vision to recognize gestures and convert them into multiple regional languages, promoting inclusivity and accessibility.

Alzheimer's Detection

This project focuses on detecting Alzheimer's disease using brain CT scans. A machine learning model processes the scans and generates diagnostic reports, aiding early detection and medical assessment. The backend, developed in Node.js, handles data processing and integrates the model to provide seamless interaction with the React frontend.

Github link

CardioSense.AI

CardioSense.AI is an AI-powered heart disease prediction system that leverages machine learning to assess cardiovascular disease risk based on key health parameters. The project provides a user-friendly web-based interface where users can enter their health details and receive instant predictions about their heart disease risk.

Github link

Deepfake Detection (FakeShield)

FakeShield is a deepfake image detection system that leverages machine learning to distinguish real images from manipulated ones. Trained on a specialized dataset, the model identifies fake images based on subtle inconsistencies. The project features a frontend built with HTML, CSS, and JavaScript, while the backend includes authentication logic and database integration for secure and efficient detection.
